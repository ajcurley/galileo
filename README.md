## Galileo
This library is primarily a learning exercise in implementing geospatial algorithms in pure Go using the standard library. This library should not be used in any production-facing applications as it may not be actively maintained or stable in the future.

### Roadmap
Below is a cursory roadmap of features that are planned for implenentation. They are not listed in any particular order.
- TIFF 6.0 encoding/decoding
- GeoTIFF 1.1 encoding/decoding
- Cloud-optimized GeoTIFF 1.0 reading
- Basic 2D planar geometry types (Point, Polygon, Box, etc.)
- Basic 2D planar spatial measurement (area, length, distance, etc.)
- Basic 2D planar spatial topology (intersects, contains, overlaps, touches, etc.)
- Basic 2D planar spatial overlays (intersection, union, difference, etc.)
- Common CRS transformations (UTM, EPSG:4326, EPGS:3857, EPSG:5070, etc.)
- GeoJSON encoding/decoding
- WKB encoding/decoding
- WKT encoding/decoding
- Spatial indexing and k-nearest neighbor search (SR-Tree)
